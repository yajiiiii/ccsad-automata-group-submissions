# Automata Final Project - Group 3

## 📋 Project Description

A comprehensive web-based collection of case studies developed as the final project for the Automata subject. This project showcases fundamental algorithms and mathematical concepts through an interactive, user-friendly interface. Each tool demonstrates key principles in automata theory and discrete mathematics with step-by-step computations and detailed explanations.

## ⭐ Core Features

1. **Division Algorithm**
   - Computes the quotient and remainder for any two positive integers
   - Displays the mathematical relationship: a = b·q + r (where 0 ≤ r < b)
   - Automatically handles input ordering and validates user inputs

2. **Euclidean Algorithm**
   - Calculates the Greatest Common Divisor (GCD) step by step
   - Computes the Least Common Multiple (LCM) for two numbers
   - Provides detailed algorithmic steps for educational purposes

3. **Collatz Sequence**
   - Generates the Collatz sequence from a positive odd integer
   - Visualizes the iterative process toward convergence
   - Demonstrates dynamic system behavior and mathematical patterns

4. **Palindrome Checker**
   - Tests whether a string reads the same forwards and backwards
   - Useful for understanding string manipulation and comparison algorithms
   - Supports case-insensitive matching

5. **Recursive Sequences**
   - Generates Fibonacci, Lucas, and Tribonacci sequences
   - Computes any number of terms on demand
   - Illustrates recursive algorithms and their computational patterns

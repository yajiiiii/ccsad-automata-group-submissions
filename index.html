Division Algorithm

About

For any positive integers a and b, there exist unique integers q (quotient) and r (remainder) such that:

a = b · q + r,  0 ≤ r < b

Euclidean Algorithm

About

Repeatedly applies the division algorithm to find the GCD of two integers. The LCM follows from lcm(a,b) = (a·b) / gcd(a,b).

Collatz Sequence

About

Starting from a positive odd integer n:

if even → n / 2  |  if odd → 3n + 1

Repeat until reaching 1. The Collatz conjecture states this always terminates.
